### 订单状态机(契约原样)

```
PENDING_CONFIRM → CONFIRMED → PREPARING → READY_FOR_PICKUP
  → PICKED_UP → DELIVERING → DELIVERED → COMPLETED
PENDING_CONFIRM → CANCELLED
任意进行中状态 → REFUND_REQUEST → REFUNDED
```

## 优先级建议

- **MVP 必须**:模块 1-5, 7-9, 11-15, 17 → 共 **14 个**, ~103 个功能点(含仓库管理 + 视角切换)
- **MVP 简化做**:模块 10(退款)、16(结算)、IM 富媒体 → **3 个**, 用最短路径走通
- **MVP 后置**:模块 6(营销)、搜索、analytics、评价/收藏、电子围栏、跨仓调拨 → 等 MVP 跑起来再迭代
- **2026-06-17 新增关注点**:模块 3 的仓库管理 + 模块 17 的视角切换器是 MVP 关键新功能,W2-W3 优先实现
